# Custom levels

Agent-generated levels live here — one folder per level (`16-some-slug/`, `17-…`), auto-discovered by the game at startup. Nothing to register.

- **Generate levels:** ask your AI coding agent — see [AGENTS.md](../../../AGENTS.md) (any agent) or the bundled `bugbound-levelsmith` skill (Claude Code).
- **Factory reset:** delete the folders in here (and their entries in `../hints.json` + `SOLUTIONS.md`). The original 15 levels are never touched by generation.
